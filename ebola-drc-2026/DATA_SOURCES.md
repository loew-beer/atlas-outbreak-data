# Sources — Ebola Bundibugyo Outbreak, DRC 2026

Sources accessed on 2026-05-29 (initial scan) and 2026-05-30 (genomic data source update).

## Genomic Data Sources (added 2026-05-30)

39. **Pathoplexus -- BDBV Sequence Search** -- Primary repository for 2026 outbreak genomes (60 total BDBV sequences: 16 from 2026, 19 from 2007, 11 from 2012, plus partials). Data use: Pathoplexus "Restricted" licence (public health use; contact before publication).
    - https://pathoplexus.org/ebola-bdbv/search

40. **Virological.org -- Rapid Communication** -- "Initial genomes from May 2026 Bundibugyo Virus Disease Outbreak in the Democratic Republic of the Congo and Uganda." Describes sequencing workflow (Illumina + Twist panel, GridION), RACCOON pipeline (MAFFT + IQ-TREE2, HKY+gamma), and preliminary phylogenetic placement. 13+ genomes across 3 updates (May 18-28, 2026). Authors: Amuri-Aziza et al. (INRB); Dept. National Health Laboratory (Uganda). Lead contacts: Prof. Placide Mbala-Kingebeni (INRB), Dr. Isaac Ssewanyana (CPHL Uganda).
    - https://virological.org/t/initial-genomes-from-may-2026-bundibugyo-virus-disease-outbreak-in-the-democratic-republic-of-the-congo-and-uganda/1032

41. **Nextstrain -- ebola/bdbv** -- Interactive phylogenetic visualization combining historical (2007, 2012) and May 2026 outbreak genomes. Used for cross-validation of phylogenetic placement, not as sole primary data source.
    - https://nextstrain.org/ebola/bdbv

42. **NCBI GenBank / RefSeq** -- Stable source for historical reference genomes. Key accessions: NC_014373.1 (2007 Uganda RefSeq reference), FJ217161.1 (2007 Uganda complete genome), KC545393.1-KC545396.1 (2012 DRC complete genomes). 34 total historical sequences used in local analysis.
    - https://www.ncbi.nlm.nih.gov/nuccore/?term=Bundibugyo+ebolavirus

43. **INRB-UMIE GitHub -- Ebola_DRC_2026** -- Epidemiological analysis repository (population, mobility, health facility data). Directs to Pathoplexus for genomic sequences. MIT licence for code. Last updated 2026-05-29.
    - https://github.com/INRB-UMIE/Ebola_DRC_2026

Note: **GISAID** was evaluated but no confirmed BDBV records were found. It is not listed as a primary BDBV sequence source.
